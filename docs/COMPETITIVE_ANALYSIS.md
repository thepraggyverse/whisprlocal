## Wispr Flow (iOS)
Category: AI Voice Keyboard
Core offering: voice-first keyboard extension that transcribes + polishes in real-time inside any app. Paid subscription.
Where WhisprLocal differs: fully local (no cloud inference). Open source. Free.
Relevant features we should ship: keyboard extension (M4), AI polish (M3), per-app context awareness (post-v1.0).

## Raycast (iOS)
Category: Productivity command palette with voice
Core offering: snippets, AI commands, dictation, shortcuts — all accessible via keyboard surface.
Where WhisprLocal differs: voice-first (not command-first). No cloud AI. Single-purpose rather than Swiss Army knife.
Relevant features we should ship: snippet expansion (M5), custom dictionary (M5), voice shortcuts (M5).

## Ghost Pepper (macOS)
Category: Power-user voice-to-text with model flexibility
Core offering: macOS menu-bar app. Multi-engine STT (Whisper + Parakeet + Qwen-ASR). Multi-size cleanup models (Qwen 3.5 in 0.8B/2B/4B). Context Bundler for screen-content capture. Meeting transcript auto-detection.
Where WhisprLocal differs: mobile-first (iOS, not macOS). Curated model catalog. Simpler UX for non-developers.
Relevant features we should ship: meeting transcription mode (post-v1.0), context-aware polish (post-v1.0).

## Positioning summary
WhisprLocal is the privacy-verifiable mobile voice keyboard.
- Primary axis: privacy (100% local, open source, MIT)
- Secondary axis: mobile-first (vs Ghost Pepper's macOS focus)
- NOT competing on: model variety (vs Ghost Pepper), command-palette breadth (vs Raycast)
- App Store copy lead: "Your voice, your device, your control."
